Politica de confidentialitate

Aceasta politica explica in mod transparent ce date pot fi procesate cand este accesat ghid-rta.ro sau cand este folosita pagina Facebook Ghid RTA MTL - Smokee.

Versiune: 21 iulie 2026Aplicatie informativa 18+Fara cont de utilizator

1. Administrator si contact

Serviciul este administrat sub denumirea Ghid RTA MTL - Smokee. Pentru intrebari privind confidentialitatea sau exercitarea drepturilor, contactul se face prin pagina oficiala Facebook Ghid RTA MTL - Smokee.

2. Ce date sunt procesate

  • Ghidul nu solicita crearea unui cont, nu primeste plati si nu contine formular de transmitere directa a datelor personale.
  • Furnizorii tehnici pot procesa automat adresa IP, tipul browserului, dispozitivul, pagina solicitata, data si ora, exclusiv pentru livrare, securitate si diagnostic tehnic.
  • Limba selectata, acceptul 18+ si copiile temporare ale catalogului pot fi pastrate local in browser, fara a crea un profil personal pe serverul ghidului.
  • Daca este acordata permisiunea de localizare, coordonatele sunt folosite pentru afisarea vremii locale si sunt transmise catre Open-Meteo. Ghidul nu le salveaza intr-o baza proprie.
  • Contorul public de vizite este furnizat de visitor-badge.laobi.icu si poate primi datele tehnice asociate unei cereri web.
  • Aplicatia Meta este folosita de administratorii paginii pentru publicarea de continut. Vizitatorii ghidului nu se autentifica prin aceasta aplicatie si nu sunt profilati de ghid.

3. Scopurile procesarii

Datele tehnice sunt folosite pentru functionarea si securitatea site-ului, pastrarea preferintelor locale, afisarea informatiilor solicitate, masurarea agregata a accesului si publicarea continutului paginii Facebook.

4. Servicii externe

Ghidul poate folosi sau trimite catre GitHub Pages, Cloudflare, Open-Meteo, visitor-badge.laobi.icu, Facebook, YouTube si Smokee.ro. La deschiderea unui link extern se aplica politica serviciului respectiv. Ghidul nu vinde si nu inchiriaza date personale.

5. Pastrarea datelor

Datele locale raman in browser pana la stergerea lor de catre utilizator. Jurnalele tehnice si datele procesate de furnizorii externi sunt pastrate conform politicilor si termenelor proprii ale acestora.

6. Drepturile persoanei vizate

In conditiile Regulamentului general privind protectia datelor, pot fi solicitate informatii, acces, corectare, stergere, restrictionare sau opozitie. Instructiunile practice sunt disponibile pe pagina Stergere date.

7. Continut destinat adultilor

Ghidul este destinat exclusiv persoanelor de cel putin 18 ani. Nu se urmareste colectarea intentionata a datelor minorilor.

Privacy Policy

This policy explains what data may be processed when ghid-rta.ro or the Ghid RTA MTL - Smokee Facebook Page is used.

Controller and contact

The service is operated as Ghid RTA MTL - Smokee. Privacy requests can be sent through the official Facebook Page.

Data and purposes

No visitor account, payment or direct personal-data form is provided. Hosting and security providers may process standard technical request data. Language, age confirmation and temporary catalogue data may be stored locally in the browser. With browser permission, location coordinates are sent to Open-Meteo to display local weather and are not retained in the guide's own database. The public visitor badge may process request-related technical data.

Meta application

The Meta application is used by Page administrators to publish Page content. Website visitors do not sign in through the application and the guide does not create visitor profiles from Facebook data.

External services and rights

The service may interact with GitHub Pages, Cloudflare, Open-Meteo, visitor-badge.laobi.icu, Facebook, YouTube and Smokee.ro. Their own policies apply when those services are accessed. Applicable privacy rights can be exercised through the contact above. See Data deletion instructions.
