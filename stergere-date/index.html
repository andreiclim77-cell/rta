Stergerea datelor

Ghidul nu creeaza conturi pentru vizitatori si nu pastreaza o baza proprie cu profiluri personale. Datele locale pot fi eliminate direct din browser.

Date pastrate in browser

Deschide setarile browserului.
Cauta sectiunea Confidentialitate, Date site sau Cookies si date site.
Cauta ghid-rta.ro.
Selecteaza datele asociate domeniului.
Sterge datele site-ului.
Vor fi eliminate limba aleasa, acceptul 18+ si copiile locale temporare ale catalogului.
Revoca localizarea, daca este cazul.
Din permisiunile site-ului, seteaza Location sau Locatie pe Block/Blocheaza.

Data deletion instructions

The guide does not create visitor accounts or maintain its own personal-profile database. To remove local data, open the browser's site-data settings, select ghid-rta.ro, clear its stored data and revoke Location permission if previously granted.

The Meta application is used for Page administration and publishing. Regular visitors do not sign in through it. If access was granted, remove Ghid RTA MTL from Facebook Settings, Apps and Websites. A deletion or access request may also be sent by private message to the official Facebook Page with the subject Data deletion request.
